# Visualizing-Airbnb-Data
This project is a full data analysis of Airbnb data. The project started out from sourcing the CSVs, to inspecting and cleaning the data, to exploring and summarizing data, visualizaing data, and then finally making a business report. Before working on the project I scoped out required steps and came up with business problems to solve. 

The three main business problems are:
1. Which property types appear to have the strongest demand based on review volume?
2. What listing characteristics are associated with higher guest demand?
3. How does pricing influence occupancy?

The business report includes all information about what conclusions were made and the process of doing so. Though if you want to view the code, I provided the code as well. I also left comments in the code as well, though the code is quite self-explanatory.
